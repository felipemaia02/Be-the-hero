# Be-the-hero

Para iniciar o servidor Express, deve entrar na pastas do backend e usar o seguinte comando:
```
npm install
```
e logo em seguida depois de instalar usar o seguinte comando:
```
npm start
```

para iniciar o frontend realizar o mesmo processo, ir na pasta do frontend e usar o seguinte comando:
```
npm install
```
e logo em seguinda depois de instalar usar:
```
npm start
```
dai é só acessar o http://localhost:3000


A parte mobile foi desenvolvida usando o expo, então para rodar o aplicativo no celular ou em algum emulador, precisa instalar o expo na sua plataforma mobile.

Após a instalação do expo é só usar o mesmos comandos que foram utilizados no Backend/frontend

<h3>Lembrando para que a aplicação funcionar sem problemas, deve executar o servidor backend.</h3>
